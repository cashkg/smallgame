# 🏆 經典競技場 (Classic Arena)
### 專為專業玩家打造的跨平台、免下載休閒競技平台

這是一套基於 **HTML5 / JavaScript** 開發的模組化遊戲系統，結合了 **LINE Login (LIFF)** 身分識別與 **Google Sheets** 雲端排行榜。專為追求「邏輯極限」與「公平競技」的資深玩家設計。

---

## 🚀 核心特色

### 1. 雙天梯競技系統
每個遊戲均設有兩套獨立排行榜，滿足不同類型的競技愛好：
* **規格賽 (格數/模式榜)**：同規格下的純速度對決，考驗瞬間反應與熟練度。
* **巔峰賽 (綜合積分榜)**：採用難度權重計分（$Score = \frac{f(\text{Difficulty})}{\text{Time}}$），鼓勵挑戰極限難度。

### 2. 真實身分與社交
* **LINE 帳號綁定**：自動抓取 LINE 暱稱與頭像，確保排行榜真實無誤。
* **一鍵報戰績**：生成專業格式的戰績報表，快速分享至 LINE 群組。
* **人氣統計**：各遊戲入口即時顯示「累計遊玩人數」，打造熱絡競技氛圍。

### 3. 公平競技保障 (Anti-Cheat)
* **失焦暫停**：切換視窗或關閉螢幕時，系統自動觸發暫停。
* **暫停遮罩**：手動或自動暫停時，盤面完全遮蓋，杜絕「暫停思考」作弊。
* **每日同題**：全台玩家使用相同日期種子（Daily Seed），確保公平競爭環境。

## 🎮 遊戲項目與規格

### 🧩 九宮格神算 (Sudoku)
* **難度階梯**：
    * 初級/中級/高級：固定挖空 35 / 42 / 49 格。
    * 極限天梯：**50 ~ 64 格個位數自選**。
* **特色**：保證唯一解，極限格數挑戰積分權重呈指數級成長。

### 🃏 經典疊牌 (Solitaire)
* **模式**：翻 1 張 / 翻 3 張 / 限輪數（3輪）。
* **計分**：綜合「耗時」與「移動步數」，懲罰無效移動，追求極致效率。

## 🛠️ 技術架構

* **前端**：Vanilla JS (ES6+), CSS3 (Flex/Grid), HTML5。
* **後端**：Google Apps Script (GAS)。
* **資料庫**：Google Sheets API。
* **驗證**：LINE Front-end Framework (LIFF)。

---

## 📂 檔案目錄說明

* `index.html`: 動態大廳入口，預留 6 個擴充插槽。
* `style.css`: 競技場深色專業主題樣式。
* `/js/auth.js`: LINE 身分驗證模組。
* `/js/app.js`: 核心控管模組 (防作弊、計時、資料交換)。
* `/js/sudoku.js`: 數獨生成與校驗模組。
* `/js/solitaire.js`: 接龍發牌與步數邏輯模組。
* `/js/ui.js`: 介面渲染與分享文字格式化。

## 📈 排行榜欄位說明 (Google Sheets)

| 欄位 | 說明 |
| :--- | :--- |
| Timestamp | 遊玩時間戳記 |
| LineID | 玩家唯一識別 (LIFF 取得) |
| Name | LINE 暱稱 |
| Game | 遊戲類型 (sudoku/solitaire) |
| Mode | 難度規格 (如: 64_extreme) |
| Time | 完成秒數 |
| Steps | 總移動步數 |
| Score | 最終換算積分 |
| Avatar | LINE 頭像網址 |
